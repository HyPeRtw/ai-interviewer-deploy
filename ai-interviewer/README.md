# AI Interviewer Helper (Parakreet Replica)

## Overview
Real-time audio transcription during interviews/meetings (Zoom/Teams via browser mic/tab share) + instant Gemini AI help.

## Local Setup
1. Client: `cd client && npm install && npm run dev`
2. Server: `cd server && npm install && npm start` (use your Gemini API key)

## Features
- Live transcription tab with start/stop/clear.
- AI Help tab for queries/feedback.

## Deployment
- Client: Vercel (connect GitHub repo)
- Server: Render (free tier)

## Usage in Meetings
- Open in Chrome.
- Share tab audio or mic in Zoom/Teams.
- Paste transcript to AI tab for instant responses.

Gemini API key input in UI (1.5 Flash recommended for speed).
